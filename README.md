# js-unit-testing-with-mocha
JavaScript Unit Testing with Mocha

1.  Init project with *npm*
    ```bash
    npm init [-y]
    ```

2.  Install *'mocha', 'chai', 'nyc', 'sinon'* and *'sinon-chai'* as development dependencies
    ```bash
    npm i -D mocha chai nyc sinon sinon-chai
    ```

3.  Install *'big-js'*
    ```bash
    npm i big-js
    ```

4.  Install *'axios'*
    ```bash
    npm i axios
    ```
